Snaek

A Snake Game with Level Editor

in C++/SDL

Snaek is a snake game loosely based on NIBBLES.BAS

Navigate the level collecting the "food" but don't run into walls, or yourself!

Fully-Featured GUI based level-editor

Snaek was coded in C++, using SDL as a way to learn the basics of SDL

Snaek features a 'standard' empty arena, but supports loading of levels encoded using a companion GUI-based level-editor program also written in C++ using SDL

Reflection

Snaek was a project to learn my way around the basics of the SDL (Simple Directmedia Layer) library

The colors and graphical style are based on NIBBLES.BAS, a "classic" QBasic program included with many PCs in the early 90s that I grew up with.

The levels are stored in an ASCII-Plaintext format and are grid-based, making a level editor an obvious candidate for inclusion.

Having a GUI-based level-editor made this project much more versatile, and helped show me the power of designing tools to use yourself to assist in the development of a game.
