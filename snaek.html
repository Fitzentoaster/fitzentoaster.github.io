Snaek(sic)

A Snake Game with Level Editor

in C++/SDL

Snaek(sic) is a snake game loosely based on NIBBLES.BAS

Navigate the level collecting the "food" but don't run into walls, or yourself!

Fully-Featured GUI based level-editor

Snaek(sic) was coded in C++, using SDL as a way to learn the basics of SDL

Snaek features a 'standard' empty arena, but supports loading of levels encoded using a companion GUI-based level-editor program also written in C++ using SDL

Reflection

Snaek(sic) is a project I developed to familiarize myself with the SDL(Simple DirectMedia Layer) library. Inspired by NIBBLES.BAS, a classic QBasic program from the early '90s that I fondly remember, the game features a colorful graphical style reminiscent of that era.

Levels are designed in a grid-based ASCII plaintext format, which led naturally to the inclusion of a GUI-based level editor. This tool significantly enhanced the project's versatility and underscored the value of creating custom development tools to streamline the design process.
