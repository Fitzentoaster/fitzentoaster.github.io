JS GoL

A Customizeable Game of Life implementation

in Javascript

JSGoL is a user-friendly, meant-to-be fun implementation of Conway's Game of Life

Random boards, or make your own, change sizes, colors, all sorts of customization.

It's all here

JSGoL was coded in Vanilla Javascript using HTML Canvas

Feature-list:

  • Change grid size
  • Adjust node colors
  • Turn on/off Grid Display
  • Change speed in Generations per second

Reflection

Repo available: Click Here

Try it out: Click Here to Try it

Made in 2023, JSGoL was an attempt by me to brush up on my Javascript skills and have some fun. I hadn't worked in Javascript in years prior, but had learned the basics before. This was a fun way to work with it and brush up on the skills.

I kept adding features until I felt it's pretty well 'complete:' this was also a bit of a workout for CSS to make it responsive and work on a variety of devices: it scales to various mobile devices easily and simply.

The Game of Life if you're not familiar is a mathematical 'game' developed by John Conway.
