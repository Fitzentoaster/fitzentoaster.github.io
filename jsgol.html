JS GoL

A Customizeable Game of Life implementation

in Javascript

JSGoL is a user-friendly, customizeable implementation of Conway's Game of Life

Random boards, or edit/make your own

Customize colors, cell size, and animation speeds

JSGoL was coded in vanilla Javascript using HTML Canvas for rendering

Feature-list:

  • Change grid size
  • Adjust node colors
  • Turn on/off Grid Display
  • Change speed in Generations Per Second

JSGoL, created in 2023, was an effort to refresh my JavaScript skills while having some fun. Having not worked with JavaScript for several years, I wanted a practical project to re-engage with the language.

I continually added features and revised the programming until I was satisfied with the end result. Additionally, this project served as a valuable exercise in CSS as I had to ensure the application was responsive and adaptable across a variety of devices and platforms, including mobile.

For those unfamiliar, the Game of Life is a mathematical "game" developed by John Conway, serving as both a fascinating model of cellular automata and a compelling programming challenge. Through this project I gained significant experience in responsive design and optimzation across different devices, making it a rewarding endeavour in my development journey.
