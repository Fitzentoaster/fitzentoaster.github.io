               <p class="text-center">In my computer architecture course, we studied the MARIE architecture as a learning experience 

to see how computer architectures could work. I programmed a fully feature simulator that can run MARIE programs,

but that wasn’t enough for me, so I designed an assembly language and assembler for it as well.

               </p>

               <p class="text-center">It was fun trying to do, for instance, integer division in a language that only has 13 instructions! It

taught me a lot about how the guts of computers work, and was also a fun way to flex my C++ muscles a bit as well.

               </p>
