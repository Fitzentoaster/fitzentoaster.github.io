Blac++Jack

A Blackjack Playing/Strategy Trainer

in C++

Blac++Jack is a Blackjack playing program with strategy teaching features

Helps train you on basic blackjack strategy and card-counting

Supports multiple card-counting systems

Play 6-deck Blackjack against your computer with optional optimal play-hints

Blac++Jack was coded in C++11

Supports eight different card-counting systems.

Six-Deck shoe simulation.

Reflection

Blac++Jack was the first project I worked on, on my own, using C++.

It was very useful in learning about C++ data structures, syntax, and OOP functionality.

I've always held an interest in gambling theory, and a blackjack training tool was a no-brainer. Supports 8 popular card-counting systems and optionally suggests optimal basic strategy plays based on the player's current situation. A graphical version is in the works using SDL.

The card and deck classes are generic enough to be used for different games: I've implemented Video Poker as well.
